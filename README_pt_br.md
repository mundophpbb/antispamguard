# ??? AntiSpam Guard

## ???? Portuguęs

### ? O que é?
O **AntiSpam Guard** é uma extensăo para phpBB que protege seu fórum contra spam de forma **invisível**, sem uso de CAPTCHA tradicional.

Ele utiliza técnicas modernas como HoneyPot, análise de comportamento, filtros de conteúdo e controle por IP.

---

### ?? Pra que serve?
Evitar:

- registros automatizados
- posts de spam
- mensagens privadas maliciosas
- abuso em formulários de contato

Tudo isso sem impactar usuários reais.

---

### ?? Como funciona?

A extensăo utiliza múltiplas camadas de proteçăo:

- ?? HoneyPot (campo invisível)
- ?? Análise de tempo de envio
- ?? Controle por IP (whitelist/blacklist)
- ?? Filtro de conteúdo (palavras e URLs)
- ??? Modo discreto
- ?? Modo simulaçăo
- ?? Logs e estatísticas
- ?? Limpeza automática via cron

---

### ?? Recursos

- Proteçăo em:
  - Registro
  - Postagens
  - Mensagens privadas
  - Formulário de contato

- Dashboard visual no ACP
- Exportaçăo CSV
- Configuraçăo avançada
- Multilíngue

---

### ??? Instalaçăo

1. Envie os arquivos para:
   `/ext/mundophpbb/antispamguard`
2. Ative no ACP
3. Configure conforme necessário

---

### ?? Recomendaçăo

Para melhor proteçăo:

- ative HoneyPot + tempo mínimo
- utilize limite por IP
- teste com modo simulaçăo antes

---

### ?? Status

Versăo atual: **2.4.1**  
Pronto para produçăo.

## ???? English

### ? What is it?
**AntiSpam Guard** is a phpBB extension that protects your forum against spam using **invisible techniques**, without traditional CAPTCHA.

It uses modern methods like HoneyPot, behavioral analysis, content filtering, and IP control.

---

### ?? What is it for?
To prevent:

- automated registrations
- spam posts
- malicious private messages
- abuse in contact forms

All without affecting real users.

---

### ?? How does it work?

The extension combines multiple protection layers:

- ?? HoneyPot (hidden field)
- ?? Submission timing analysis
- ?? IP control (whitelist/blacklist)
- ?? Content filtering (words & URLs)
- ??? Silent mode
- ?? Simulation mode
- ?? Logs and statistics
- ?? Automatic cleanup via cron

---

### ?? Features

- Protection for:
  - Registration
  - Posts
  - Private messages
  - Contact form

- ACP dashboard
- CSV export
- Advanced configuration
- Multilingual

---

### ??? Installation

1. Upload to:
   `/ext/mundophpbb/antispamguard`
2. Enable in ACP
3. Configure as needed

---

### ?? Recommendation

For best protection:

- enable HoneyPot + time check
- use IP limits
- test using simulation mode first
