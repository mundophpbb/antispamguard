# AntiSpam Guard

## ???? English

### ? What is it?
**AntiSpam Guard** is a phpBB extension that protects your forum against spam **invisibly**, without using traditional CAPTCHAs.

It uses modern techniques such as HoneyPot, behavioral analysis, content filters, and IP control.

---

### ?? What is it for?
It prevents:

- Automated registrations
- Spam posts
- Malicious private messages
- Abuse of contact forms

All without affecting legitimate users.

---

### ?? How does it work?

The extension uses multiple layers of protection:

- ??? **HoneyPot** (hidden field)
- ?? Submission timing analysis
- ?? IP control (whitelist/blacklist)
- ?? Content filtering (words and URLs)
- ?? Silent mode
- ?? Simulation mode
- ?? Logs and statistics
- ?? Automatic cleanup via cron

---

### ? Features

- Protection on:
  - Registration
  - Posting
  - Private messages
  - Contact form

- Visual dashboard in the ACP
- CSV export
- Advanced configuration
- Multilingual support

---

### ?? Installation

1. Upload the files to:  
   `/ext/mundophpbb/antispamguard`
2. Enable the extension in the ACP
3. Configure according to your needs

---

### ?? Recommendation

For maximum protection:

- Enable HoneyPot + minimum time check
- Use IP-based limits
- Test everything in **simulation mode** first
